Value Portfolios

This project (currently in progress) is a value factor-based portfolio construction tool I developed as an intern at OmniScience Capital that allows users to analyze stock portfolios based on 70+ different financial metrics and ratios. Users can rank consituent stocks of an index, sort them by quintiles or deciles based on the metric, perform backtesting, and visualize cumulative returns of these portfolios over time. The code allows for flexible start and end dates and can create value-based portfolios on the NASDAQ 100, S&P 500, and Russell Top 200 indices. The tool also provides detailed information about the tickers in each portfolio group, including their performance in the period and the distribution of the chosen metric within the portfolio. The data ranges from `2000-06-01` to `2024-06-01`. Historical index constituents, fundamental metrics and ratios, and price data was acquired from the Center of Research in Security Prices (CRSP) and Compustat databases via Wharton Research Data Services.

Features

  • Rank stocks based on a chosen financial metrics
  • Choose from the NASDAQ 100, S&P 500, and Russell Top 200
  • Perform backtesting on portfolios within a given date range
  • Visualize cumulative returns over time and other portfolio metrics
  • Detailed portfolio analysis including tickers, returns, and financial metrics
  • Perform univariate and multivariate linear regressions of returns on value factors

Regressions

The file regression.py allows users to perform regressions of returns of index constituents on chosen value factors to determine which factors, and to what degree, play a role in forecasting stock returns. The below graph and table shows the result of regressing stock returns in the NASDAQ 100 in the past year on Net Profit Margin, demonstrating that Net Profit Margin is a statistically significant factor in determining the returns on NASDAQ 100 constituents in the past year.
